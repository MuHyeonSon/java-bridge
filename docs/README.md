# 다리 건너기 게임

##  기능 구현 목록

-  입력
- [x] 다리의 길이 입력 받기 기능 구현
- [ ] 사용자에게 이동할 칸 (U or D) 입력 받기
- [ ] 게임 재시작/ 종료 여부 (R or Q) 입력 받기

- 출력
- [ ] 게임 시작시 시작 문구 출력
- [ ] 현재까지 이동한 다리의 상태를 정해진 형식에 맞춰 출력
- [ ] 게임 최종 결과를 출력 (성공여부, 시도횟수)

- 예외처리
- [x] 다리의 길이 입력 예외처리(숫자여부, 길이제한)
- [x] 이동할 칸 입력 예외처리(U와 D 중 하나의 값이 입력됐는지 여부)
